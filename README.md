# AVA - Ambiente Virtual de Aprendizagem

## 📖 Sobre o Projeto

O **AVA (Ambiente Virtual de Aprendizagem)** foi desenvolvido para centralizar treinamentos, manuais, procedimentos e materiais de apoio relacionados ao sistema ERP da empresa.

O objetivo é facilitar o acesso ao conhecimento, padronizar processos internos e acelerar o treinamento de novos colaboradores.

---

## 🎯 Objetivos

* Centralizar conteúdos de treinamento.
* Disponibilizar manuais e procedimentos operacionais.
* Facilitar a capacitação de usuários do ERP.
* Reduzir o tempo de aprendizado de novos colaboradores.
* Preservar o conhecimento organizacional.

---

## 💻 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/jucebrasil/AVA.git
```

2. Acesse a pasta do projeto:

```bash
cd AVA
```

3. Abra o arquivo `index.html` em seu navegador.
